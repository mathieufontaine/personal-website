About Me

Web developer & Digital Marketer

I love designing, building & promoting nice websites and web applications

"I am Mathieu and I love building nice websites and web applications"

Web Developer

I am a web developer. I learned fullstack development at Le Wagon in 2019 (Ruby/JS/React..). Since the end of the Bootcamp I practiced a lot by building various websites. I also acquired new skills on different technologies (back and front), learning by myself.

Passionate about Frontend

What I enjoy the most about coding is the ability to instantly see the results of my work as well as the interaction with the user. Thus, I decided to specialize in frontend development. I focus on writing accessible HTML, using modern CSS practices and clean Javascript code. I also use ReactJS & React Native libraries.

Focus on UI/UX Design

The main objective of any of my web projects is to enhance the User Experience. Exploring Websites or using Web Apps should be enjoyable and completely effortless for everybody. With my background in Marketing I naturally think about the user first when I design or develop a new project.

Working Remotely

I am currently based in Paris but I exclusively work remotely. I am much more productive when I work in my own environment. When I am not coding you will find me outdoors. I love exporing new places by foot or with my bike.
