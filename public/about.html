Web Developer

I am a web developer. I learned fullstack development at Le Wagon in 2019 (Ruby/JS/React..). Since the end of the Bootcamp I practiced a lot by building various websites. I also acquired new skills on different technologies (back and front), learning by myself.

Passionate about Frontend

What I enjoy the most about frontend developement is the interaction I create with the user, through a computer. I like to transform code into a smooth web experience that will allow owner and customers to achieve their objectives. I focus on writing accessible HTML, using modern CSS practices and clean Javascript code. I also use ReactJS & React Native libraries.

Focus on UI/UX Design

The main objective of any of my web projects is to create a smooth and effortless experience for the user. Exploring Websites or using Web Apps has to be be straightforward and enjoyable in order for the users to engage with the company. With my background in Marketing I naturally think about the user first when I design or develop a new project.

Experienced in Marketing

Digital Marketing is a field I have graduated from in 2016 (master degree). I have worked as Online Media Executive, focusing on improving the performance of SEA campaigns for the Club Med company for almost 2 years. I have also worked for a startup in Singapore as Online Project Manager, I redesigned their website and improved the visibility and conversion rate of their website.

Working Remotely
