Web Integration

I will integrate your design in HTML CSS & JS code

A web integration is a phase that allows the final mock-up of a website or web application to be made into a web page.

If you are a designer you may need to convert your design into actual code. As a frontend developer I will create structured and efficient HTML, CSS and JS code to transform your work into a website.

If necessary I can also convert your design into a wordpress website if you need to manage your content on a regular basis.

Protocol

My web integration will follow the 5 steps below:

  • 1. Separation of contents, work on design and layout of the graphic elements

  • 2. Integration of pages in HTML5 & CSS3 according to web standards

  • 3. Addition of Javacript code for interactivity (if applicable)

  • 4. Optimization of accessibility

  • 5. Optimization of all elements related to referencing

  • 6. Check compatibility with browsers and devices

How does it work?

In order for me to code your design exactly as you conceived it, I will need a detailed PSD, Figma or Sketch file of both desktop and mobile design, includieng the required assets such as fonts, images, colors, etc..

HTML & CSS are the markup languages I will use to organize and style the layout of your design. If necessary, I will use JavaScript to add specific features and interactivity.

Once I have finished the web integration I will send you a zip file with the required HTML, CSS and JS files.

Any design to convert? Don't hesitate to contact me!
